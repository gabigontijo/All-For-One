## :rocket: Running the project
* To run the repository it is necessary to clone it, give the following command to start the project:
```
npm install
```
* To up and access the docker terminal:
```
docker-compose up -d
docker exec -it all_for_one bash
npm install
```
* Start application

```
npm start
```
* or
```
 npm run dev
```
